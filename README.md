It was around my 10th birthday (in the 90s üòÅ) I enjoyed typing my first lines of code on an [Atari ST](https://en.wikipedia.org/wiki/Atari_ST) in [GFA Basic](https://en.wikipedia.org/wiki/GFA_BASIC). I subscribed to [ST Magazine](https://fr.wikipedia.org/wiki/ST_Magazine) which talked in particular about programming, with code examples (GFA Punch) in order to learn language and techniques. I created various softwares for personal use.
